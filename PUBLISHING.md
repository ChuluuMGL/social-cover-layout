# Publishing Boundary

## Included in this repository

- Original `SKILL.md` instructions.
- Original visual-route, color-system, and quality-gate references.
- Original metadata, README files, tests, and release notes.

## Not included

- Third-party Skill source text or copied repository structure.
- Creator portraits, logos, watermarks, screenshots, fonts, or single-cover reproductions.
- Credentials, cookies, API keys, publishing accounts, or platform automation.
- Generated cover outputs whose model or source-asset rights have not been checked.

## Release checklist

- Run `npm run validate`.
- Review `git diff --stat` and confirm only intended files are included.
- Check that examples use original prompts and authorized or synthetic assets.
- Confirm the repository has the MIT license and current version metadata.
- Review generated outputs separately under the image model's terms.
- Tag the release only after the visual acceptance matrix has a passing control case.

This Skill provides planning and generation instructions. It does not make a legal determination about any particular image, font, model output, or reference asset.
