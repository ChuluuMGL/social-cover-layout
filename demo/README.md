## P0 multilingual typography

The [P0 multilingual demo](./multilingual-p0-demos/) now contains five independent cover images for Simplified Chinese, Traditional Chinese, English, Japanese, and Korean. Each one keeps the proven cover composition and changes only the language brief and supporting palette; it is not a five-language card sheet.

## Technical platform checks

The [technical platform test pack](./technical-platform-tests/) uses the same brief to check canvas ratios, cropping, and safe areas. It is intentionally not presented as a new image-model generation set, and it is not the primary visual demo.

The images show the result of the test process, not a promise of performance, click-through rate, or platform acceptance. Final publishing still requires checking the current platform uploader and any rights for people, logos, screenshots, fonts, or model outputs.
